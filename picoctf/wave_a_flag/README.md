# Wave a flag
Category: General Skill, 10 points

## Description
> Can you invoke help flags for a tool or binary? This program has extraordinarily helpful information...

## Solution
Use `-h` flag when invoking the binary, and the flag will be printed to
stdout.

```
$ chmod +x warm
$ ./warm -h
```
